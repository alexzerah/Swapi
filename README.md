# SWAPI project

## Details

This app show swapi info and use :

- React
- Node.JS
- hapi
- Docker

## How to launch

You need :

- docker / docker-compose

```zsh
## First clone the project and go into it
$ git clone https://github.com/alexzerah/Swapi.git
$ cd Swapi

## then launch Docker and install everythin
$ docker-compose build
$ docker-compose up -d
```

Then, go to localhost:3000 (it can be different for Linux) and use the app.
